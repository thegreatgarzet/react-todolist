# React Todolist

Lista de tarefas com login e registro feita com React.js, Vite, React Router DOM e CSS puro.

## Demo 
- [Acesse o projeto aqui](https://react-todolist-seven-theta.vercel.app)
- O backend esta hospedado no [Render](https://render.com), a primeira request pode demorar um pouco até o servidor iniciar.

## Funcionalidades
- Login e Registro com validação
- CRUD e alteração do status de "completo" das tarefas
- Filtragem de tarefas por status, ordem alfabética e por texto
- Atenticação via JWT

## Como rodar localmente:
- cd react-todolist
- npm install
- npm run dev

## Backend
Você pode utilizar um backend próprio, mas se quiser testar com um já existente, utilize o meu ([https://github.com/thegreatgarzet/react-todolist](https://github.com/thegreatgarzet/todolist-backend))

## Variaveis de ambiente necessária
- VITE_API_URL=http://localhost:3000
